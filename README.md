How to run:  
1.  Install `miniconda` for Python 3
2. `chmod +x Miniconda3-latest-<OS>.sh`
3.  Create `conda` enviroment `conda env create -f enviroment.yml`
4.  `conda activate lhc-olympics`
5.  `jupyter-notebook`
